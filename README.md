# Cursor-Enjoy

## 介绍

Cursor-Enjoy 是一款无限使用 Cursor 授权工具，完全支持 Windows 和 MacOS 系统。

## 更新日志

- **2025-05-31**: 系统全面升级为 1.6 版本，大幅提升授权稳定性

## 功能特点

- 无限制使用 Cursor 编辑器的高级功能
- 跨平台支持 (Windows & MacOS)
- 稳定可靠的授权机制
- 定期更新维护
